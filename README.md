# PIDAS Selection Announcement Website

A simple web-based announcement system for **PIDAS Extracurricular Recruitment**. Participants can enter their **Student ID (NIS)** to check whether they have successfully passed the interview selection and view their assigned division.

## ✨ Features

- Search result using Student ID (NIS)
- Displays:
  - Participant name
  - Selection status
  - Assigned division (if accepted)
- Responsive design for desktop and mobile
- Glassmorphism UI with animated transitions
- No database required (uses a local JavaScript data file)

## 📁 Project Structure

```
.
├── index.html          # Input page
├── pengumuman.html     # Announcement page
├── style.css           # Styling
├── script.js           # Application logic
├── data.js             # Participant data
├── assets/
│   ├── logo.png
│   └── background.jpg
└── README.md
```

## 🚀 How to Run

1. Download or clone this repository.
2. Open `index.html` in your browser.
3. Enter a valid Student ID (NIS).
4. The system will display the participant's selection result.

> No installation or backend server is required.

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla)

## 📌 Notes

- Participant data is stored in `data.js`.
- Accepted participants will see their assigned division.
- Student IDs that are not found will be treated as **not accepted**.

Developed for the **PIDAS Extracurricular Recruitment Selection**.
